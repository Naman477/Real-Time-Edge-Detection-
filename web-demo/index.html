Real-Time Edge Detection

Web-based demo using OpenCV.js

Please allow camera access when prompted. You should see the edge-detected video below.

Note: For best results, ensure you're using a modern browser with camera support.

About This Demo

This web-based edge detection demo uses OpenCV.js to process your camera feed in real-time. The Canny edge detection algorithm identifies edges in the image by looking for areas of rapid intensity change.

How It Works

  1. Your browser accesses your camera (with your permission)
  2. Each video frame is processed using OpenCV's Canny edge detection
  3. The processed image is displayed in real-time
